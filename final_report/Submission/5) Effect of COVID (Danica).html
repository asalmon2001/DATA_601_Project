Guiding Question 5 (Danica): How did COVID Affect Building Permit Types and Work Class Groups in Calgary?¶

Domain¶

COVID impacted the world both socially and economically. Industry halted, employees were furloughed, and remote working became the norm. This question will delve into the effects on building permit types. Was there an increase in residential permits or non-residential permits? Were the work class types different? Demolition, residential improvement or new start? This project will investigate the unique effects of a COVID related world shut down on the microcosm of construction in Calgary.

Data Set¶

The data used in the exploration of this guiding question was categorical data exported from the Building Permits from Open Calgary, of which the terms of use indicate that the data is in a CSV format (The City of Calgary, 2023). The data was exported four times. The data was filtered for the years 2019, 2020, and 2021 individually and then for the years 2019-2021 collectively. This was done to ensure that there were no errors when doing the visualizations and the values could be cross-referenced with the year specific data. Once the data was exported to CSV it was read into Jupyter using as a Pandas data frame. Then the data was checked for null values, of which there were none. Once the data was cleaned, there were differing next steps for the data crossing all years versus the data that was focused on one year.

One of the challenges with the creation of visualizations was determining which column contained the best data for the analysis. There were several columns with similar categorical data inputs and we needed to determine which would be the simplest for all audience members. Work Class Grouped, broken up into 4 categories:

  • Improvement: an addition or change to existing structures
  • New: a new structure that is separate from existing structures
  • Demolition: destruction or removal of a structure
  • Unspecified: does not fall into the other categories (ie - fire alarm installs)

The other column that was selected for this analysis was the Permit Class Mapped, which is broken up into 3 categories:

  • Residential: for living or housing purposes
  • Non-residential: for commercial use
  • Unspecified: what does not fit into the other categories

This column was selected over other columns with similar information due to the simplicity of categorization (using mostly commonly used terms) and the ability to visualize the data.

Purpose¶

The following section breaks the data up into their respective years (pre-covid, covid, and post-covid) for 2019,2020 and 2021. The Process¶

The data was filtered for AppliedDate, PermitClassMapped, WorkClass, WorkClassGroup, and PermitType. This keeps the columns we want and gets rid of the excess 25 columns. Then we checked for null or NaN values and grouped by Work Class Group and Applied Date for the frequency by group of permit visualization and filtered by Permit Class Type and Applied Date for the frequency by type visualization. Once the data was grouped and the values were counted by category. We used Matplotlib to create the bar chart for each year and each variable (WorkClassGroup and PermitClassMapped). To ensure the data as callable and the visualizations could be effectively plotted, we had to add .reset_index() while grouping so we could call columns by index. To ensure the visualizations were accurate, we displayed the dataframe after it was grouped and compared it to the bar graph values.

Key Takeaways from the year specific Visualizations¶

Each of the Permit Type visualizations showed that the "Improvement" work class had the highest volume of permits followed by "New", and that Residential permits were the most common permit type. The next step will show the trends over the years and determine if there was a noticeable change in the ratios between Residential permit types and Improvement work permit class, or if the ratios remained fairly constant across the years. It is difficult to make substantial conclusions from the individual year aside from direct comparison of Permit Type of Work Class Group. The proportions year by year remain the similar and there is more value to this question to bring them together across the time period, as done in the next part.

Trends Across All Years¶

Purpose¶

Two visualizations were created to display the trends across COVID. A Sunburst diagram that encompasses all the data to summarize the findings and then a bar plot to better show the specific Work Class Group trends across COVID.

The Process¶

The following visualizations were created by reading in the csv file for the years 2019-2021 in a Pandas data frame then filtering for AppliedDate, PermitClassMapped, WorkClass, WorkClassGroup, and PermitType. This keeps the columns we want and gets rid of the excess 25 columns. Then the data was checked for null or NaN values. Before creating the actual visualization a new column had to be added that converted the Applied Date datetime and then the year was extracted and added to a second new column. This allowed the data to be grouped by year.

To create the summary bar chart, the data was grouped by work class group and then again by applied date (to get the 3 columns). The x values were created by creating a Numpy array from 0 to length of Work Class Groups (less 1). This creates an index that can be used to call Work Class Groups. The bars were then plotted specifying the location on the x-axis using the x +/- width argument. This ensures the bar plots are not plotted on in the same place and are visible. Labels for the title, and the x/y-axis were added and the x-ticks were specified to be angled to ensure legibility.

The Sunburst plot was created using Plotly express. Prior to plotting, the value counts of each work class group was taken to help predict the overall frequency. Then the data was grouped by Year, PermitClassMapped, WorkClassGroup and the values were set by the applied date. This allowed the data to be displayed by year. Finally .reset_index() was included to ensure the columns were callable by name.

Key Takeaways and Conclusion¶

Plot: "Permit frequency by Work Class Group in 2019-2021"

  • shows the trends of Work Class permits across 2019-2021. The "Demolition" and "Unspecified" work class group remains fairly unchanged across the years thus it can be surmised that those groups remain unaffected by the lockdowns,during COVID, and the economic turn around, post COVID.
  • While the "Improvement" class remained the highest followed by the "New" class across the years, there was a noticeable increase in "Improvements' ' in 2020 and 2021. Upon research, the improvements could be attributed to the increase of home improvement projects that were initiated through lockdowns and carried on into 2021. This is supported by the data collected by Architectural Digest that discusses theincrease in home improvement spending in 2020 and 2021 (Architectural Digest, 2023).
  • The last takeaway from this graph is that there was a large increase in the "New" class in the year 2021. This may be attributed to pent up demand and limitations due to lockdowns in 2019 that may have caused the large increase in permits (T.C. Press, 2023).

Plot: "PermitClassMapped for 2019-2021"

  • This plot shows fairly steady proportions of permit types across the years with residential remaining the largest proportion. The changes from 2019-2021 are not well visualized in this graph without hovering over the plot to get the number of permits. This plot is useful for a quick understanding of proportion, but less so when looking at trend changes.

Plot: Sunburst Diagram

  • This plot encompasses all the data and gives an interactive overview of the data. The impact of COVID on workclass groups paired with permit types can be seen aided by the interactivity of the diagram. This diagram better illustrates the change of permit type across the years. The size of the second ring gives a simpler indication of the change in Residential and Non-Residential building permits specifically between 2019 and 2020.

Conclusion¶

COVDE did impact Work Class Groups and Building Permit types. The overall ranking of most frequent in either category did not change between 2019 and 2021 but the movement within the Work Class Groups was significant. "Improvement" permits increased as people in Calgary were locked down in their homes and looking for activities to fill their time and when COVID began to slow down in 2021, permits for the "New" work class drastically increased as people were starting to work again and construction tried to keep up with the pent up demand from 2020.
